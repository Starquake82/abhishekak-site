# AI-Assisted BA & BI Portfolio Website

DOMAIN: Personal Portfolio / Business Analysis / Business Intelligence / AI  
TYPE: AI-Assisted Web Project  
STATUS: Live  
URL: https://abhishekak.site  
STACK: Next.js / React / TypeScript / GSAP / Lenis  

This project is a live professional portfolio for **Abhishek Anil Kininge**, built to demonstrate how structured business analysis, business intelligence experience, and AI-assisted delivery can be combined into a polished digital product.

It is not only a portfolio website. It is also an AI project: the site was developed through an AI-assisted coding workflow, with human-led requirements framing, content architecture, case-study structure, validation, and product direction.

## 01 / Project Thesis

The website translates professional experience into an interactive product.

It brings together real-world BA and BI work, process improvement case studies, portfolio projects, certification evidence, CV downloads, and project artefacts into one structured experience. The goal is to make business capability visible: not through claims alone, but through documented work, clear navigation, and evidence-led storytelling.

The project follows the personal brand principle:

> Structured Diagnosis for Business, Systems & AI.

## 02 / Why This Project Exists

This portfolio was created to serve three purposes:

- Present BA, BI, and process improvement experience through structured case studies.
- Demonstrate the use of AI coding tools to convert business intent into a working web product.
- Show how an analyst can act as product owner, content architect, reviewer, and delivery lead in an AI-assisted build.

The site is deliberately shaped like a professional artefact: clear sections, visible context, evidence-first content, and a restrained visual system.

## 03 / AI-Assisted Delivery

The website was built using an AI-assisted coding workflow, including Claude Code-style prompt-led development and iterative refinement.

The AI workflow supported:

- Translating requirements into page sections and components.
- Creating and refining React / Next.js implementation.
- Iterating on layout, animation, responsiveness, and interaction behavior.
- Structuring portfolio content into reusable data models.
- Moving from concept to deployed site faster than a traditional hand-coded workflow.

The human role remained central:

- Defining the product objective.
- Supplying domain knowledge and professional context.
- Reviewing outputs for accuracy, tone, and business relevance.
- Validating case-study structure and portfolio evidence.
- Directing the final user experience.

## 06 / Product Features

- Interactive hero section with animated canvas background.
- Smooth scrolling and custom navigation.
- Dedicated sections for about, experience, projects, certifications, education, and contact.

- Dynamic project detail pages using structured project data.
- Downloadable BA and BI CVs.

```bash

```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

Build for production:

```bash
npm run build
```

Run lint checks:

```bash
npm run lint
```

## 11 / Design Direction

The visual language follows **AK Brand OS**:

- One signal colour: orange.
- Warm paper and near-black surfaces.
- Serif headlines, sans body text, mono metadata.
- Thin rules, square edges, and restrained spacing.
- Structure before decoration.

The design intent is calm, precise, and evidence-led: closer to a professional diagnostic artefact than a marketing landing page.
